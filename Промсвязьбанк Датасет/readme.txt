1.	Результатом работы должны быть ссылка на проект на GitHub и архив с собранным приложением, готовым к использованию.
2.	Проект на GitHub должен содержать:
	•	Исходный код
	•	Readme.md файл со следующей информацией:
		o	Описание приложения (какие задачи решает и т.д.)
		o	Инструкция по запуску
		o	Руководство пользователя 
		Не обязательная, но желательная информация:
		o	Минимальные системные требования
		o	Инструкция по сборке
3.	Архив должен содержать все необходимые файлы для запуска приложения и инструкции по запуску и работе с приложением (либо ссылку на readme.md GitHub репозитория).
